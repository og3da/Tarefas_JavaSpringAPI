# Tarefas_JavaSpringAPI
API JAVA usando Spring Boot para gerenciar tarfeas 
Essa API realiza um CRUD de tarefas com as seguintes funcionalidades:
- criar tarefa
- ler tarefa especifica
- let todas tarefa
- alterar tarefa
- apagar tarefa
